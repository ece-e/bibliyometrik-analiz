# Bibliometric Analysis of Incretin Hormone Research

This repository contains a comprehensive **bibliometric analysis** of the scientific literature on *incretin hormone research*, based on data extracted from the **Web of Science (WoS)** database.

The analysis explores publication trends, journal impact, keyword evolution, collaboration patterns, and global research contributions using Python-based data analysis and visualization techniques.

---

## ğŸ“Š Scope of the Analysis

The study includes the following components:

- **Annual publication trends** (excluding incomplete year data)
- **Top journals analysis** based on:
  - Number of publications  
  - Average citation counts  
  - 5-Year Impact Factor
- **Keyword analysis**, including:
  - Most frequent author keywords
  - Temporal evolution of keywords
  - Keyword co-occurrence network analysis
- **Citation distribution analysis** by publication year
- **Country-level analysis**, including:
  - Publication volume
  - Average citations
  - 3D impact and velocity visualizations
- **Conceptual evolution stream** of key research topics
- **Global knowledge power map** using geographic visualization

Both **static (Matplotlib / Seaborn)** and **interactive (Plotly)** visualizations are used.
